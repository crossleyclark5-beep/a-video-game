# A Video Game

A tiny browser game you can play on phone or computer.

## Play (share this with your brother)

Once GitHub Pages is live:

**https://crossleyclark5-beep.github.io/a-video-game/**

Open that link on any phone or computer. No Cursor install needed.

## Work on this from anywhere in Cursor

### On another computer
1. Install [Cursor](https://cursor.com)
2. Clone or open: https://github.com/crossleyclark5-beep/a-video-game

### On your phone (build / agents)
1. Open [cursor.com/agents](https://cursor.com/agents) (or the Cursor iOS app on a paid plan)
2. Sign in with the same Cursor account
3. Choose this GitHub repo and start an agent

Phone Cursor is for directing agents and reviewing work — not a full code editor. Playing the game uses the public link above.

## Local play

Open `index.html` in a browser, or from this folder:

```powershell
start index.html
```
