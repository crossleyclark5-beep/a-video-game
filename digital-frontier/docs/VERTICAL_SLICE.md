# Grassland Vertical Slice

The first chapter should feel like a complete handheld adventure — not a prototype tour.

See also: `FIRST_CHAPTER.md` (sanctum, guidance, evolution ceremony).

## Player loop

Boot → Partner → Home care → Adventure → Explore Park → Quests → Shop → Battle → Sanctum → Boss → Evolution → Home → Autosave

## Main chapter spine

1. **First Steps** — welcome sign, chest, Park Guide  
2. **Grassland Call** — talk to Field Ranger  
3. **Pine Threat** — trail cache → Fuel Clerk → defeat **Glitch Alpha**  
4. **Hollow Challenge** — Pine Hollow → Root Gate → **Hollow Warden**  
5. **Chapter One Clear** — celebration, story echo, first growth moment
