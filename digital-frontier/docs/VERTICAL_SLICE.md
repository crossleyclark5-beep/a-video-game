# Grassland Vertical Slice

The first 30–60 minutes should feel like a complete handheld adventure — not a prototype tour.

## Player loop

Boot → Partner → Home care → Adventure → Explore Park → Quests → Shop → Battle → Level/bond → Home → Autosave

## Main chapter spine

1. **First Steps** — welcome sign, chest, Park Guide  
2. **Grassland Call** — talk to Field Ranger  
3. **Pine Threat** — defeat **Glitch Alpha** (mini-boss)  
4. **Hollow Challenge** — discover Pine Hollow, defeat **Hollow Warden**  
5. **Chapter One Clear** — title card, Bits bonus, side quests open wide  

## Polish included

- `DeviceDialogue` modal (A continue) instead of toast-only NPC talk  
- Quest-state lines via `ChapterCast`  
- Fuel Stop shop counter + clerk in Pleasant Park  
- Field Salve heals player HP in the field  
- Death reboots at Pleasant Park hub  
- Boss / mini-boss HP bars  
- Chapter title cards (once per save, not every deploy)  
- Save confirmation toast  
- Main quest prioritized on Field Unit chrome  
- Side quests drip one-at-a-time after Grassland Call  
- Pinned Field Ranger / Researcher near Pleasant Park  
- Trail lore caches toward Pine Hollow  
- Home ↔ Adventure autosave + deploy/return feedback
